﻿ OpenWebGIS: пользовательское соглашение


Администрация сайта opengis.dlinkddns.com (далее OpenWebGIS) предлагает пользователю сети Интернет (далее – Пользователь) - использовать свои сервисы на условиях, изложенных в настоящем Пользовательском соглашении (далее — «Соглашение»).

Соглашение вступает в силу с момента выражения Пользователем согласия с его условиями.

Пользователь считается принявшим условия Соглашения, если он начал использовать какой-либо сервис/его отдельные функции, либо пройдя процедуру регистрации.Для того чтобы воспользоваться некоторыми сервисами OpenWebGIS или некоторыми отдельными функциями сервисов, Пользователю необходимо пройти процедуру регистрации,в результате которой для Пользователя будет создана уникальная учетная запись. Для регистрации Пользователь должен написать письмо на адрес электронной почты OpenWebGIS@yandex.ru. Письмо может быть написано в произвольной форме при этом в нём должен быть указан логин и пароль, который Пользователь хочет использовать для доступа к своей учетной записи.

Пока автоматическая регистрация на сайте opengis.dlinkddns.com не предусмотрена. О произведённой регистрации на сайте opengis.dlinkddns.com Пользователь будет уведомлен в течении суток письмом по электронному адресу с которого был произведён запрос о регистрации

Администрация сайта вправе запретить использование определенных логинов, а также устанавливать требования к логину и паролю (длина, допустимые символы и т.д.).
