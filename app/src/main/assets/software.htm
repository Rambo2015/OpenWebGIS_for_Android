﻿ OpenWebGIS: программное обеспечение


OpenWebGIS создана на основе Открытого программного обеспечения (англ. open-source software) — программного обеспечения с открытым исходным кодом.
Для реализации этого проекта выбрано следующее програмное обеспечение:

1.Geoserver – открытый сервер, написанный на Java, для сетевого размещения пространственных данных и их обработки;
2.OpenLayers - библиотека с открытым исходным кодом, написанная на JavaScript, предназначенная для создания карт на основе программного интерфейса (API) ;
3.PostGIS - расширение объектно-реляционной СУБД PostgreSQL, предназначенное для хранения в базе географических данных ;
4.Proj4js - библиотека методов для трансформации между различными географическими координатными системами ;

5.Cesium - Cesium это библиотека JavaScript с открытым исходным кодом для создания 3D глобусов и 2D карт без браузерных плагинов. Библиотека использует технологию WebGL;

6.Политическая карта мира - взята с сайта http://www.naturalearthdata.com ;

7.Алгоритм для расчёта параметров орбиты спутника - взят у Shashwat Kandadai https://github.com/shashwatak/satellite-js

8.OSM Buildings (OSM здания). -Библиотека JavaScript для визуализации OpenStreetMap геометрии зданий на интерактивных картах.'OSM Buildings' создаёт 3D вид зданий по данным OpenStreetMap (OSM) на 2D карте. 'OSM Buildings' не создаёт настоящую 3D модель но использует специальную технологию наложения тени для того что бы изображения зданий казались трёхмерными ‘3D’.Copyright (c) 2014, OSM Buildings, Jan Marsch All rights reserved.
